# Movie App

## 概要

映画、テレビ番組データの表示

### データ取得元

- [The Movie DataBase(以下, TMDB)](https://developers.themoviedb.org/3) ☜ クリック
  - データ内容
    - ストリー
    - ポスター
    - サムネイル
    - 評価点数

#### 準備事項

- API キー発行
  - TMDB API 使用のためのキー
    - 上記リンクから会員登録・ログイン
    - 必要な情報入力後キーを取得
